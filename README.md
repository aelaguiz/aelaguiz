# Amir Elaguizy

I build AI-native companies and open-source agent infrastructure.

Right now I am focused on the boring layer that makes agents usable in real work: typed contracts, visible state, recoverability, and operator control.

## Current work

- [Doctrine](https://github.com/aelaguiz/doctrine) — typed DSL and compiler for reusable agent instructions and workflows that compile to `AGENTS.md`
- [Rally](https://github.com/aelaguiz/rally) — filesystem-first harness for repo-local multi-agent coding workflows
- [codex-autoresearcher](https://github.com/aelaguiz/codex-autoresearcher) — structured Codex-native optimization harness with separate worker and judge turns
- [aimgr](https://github.com/aelaguiz/aimgr) — operator tooling for AI accounts, browsers, and local agent rails

## What I care about

- durable agent identity
- visible state instead of hidden control planes
- repo-local memory and honest recovery paths
- tools that hold up under real operator use
